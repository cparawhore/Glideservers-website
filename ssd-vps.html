Knowledgebase

What is a VPS?

A virtual private server (VPS) is a virtual machine sold as a service by an Internet hosting service. A VPS runs its own copy of an operating system, and customers have superuser-level access to that operating system instance, so they can install almost any software that runs on that OS.

What Can I use a VPS for?

You could do so many different things with a VPS, some examples are setting up a VPN, website hosting, SSH gateway, game servers, (depending on hardware allocation) email servers and storage/database types.

Why choose VPS over shared hosting?

Having your own allocation of hardware increases stability and performance of your VPS, it also provides you with more disk space, CPU and RAM.

Does my VPS come with an IPV4 address?

Yes, all VPS's come with an individual IPV4 address, with no port restrictions. The 2048 packages come with two IP's instead of one.
